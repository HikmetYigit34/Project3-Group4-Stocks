  1. Data Collection
    Data is collected from Yahoo Finance as csv file. Data contains top 100 companies stocks for 12 months back starting from 2023 September. Resolution is one dataset (row) per day except holidays.
  2. Database
    Collected csv file from Yahoo Finance converted to xml database. Xml is a NoSQL database.
  3. Visualization
    Html, css, javascript combination.
  4. API
    Selected company and time frame dataset to be created as a json file.
